### <a id="Page"></a> Click
| 분석항목 | 키 | 설명 |
| :------ | :------ | :------ |
| 이벤트 | event | 화면명 | 
| 검색 결과 클릭수 | SCH  | 사용자가 내부 검색 결과를 보고 검색 결과별로 클릭한 횟수를 분석 |
| 내부 클릭 분석 | CKC | 사용자가 내부 검색 결과를 보고 검색 결과별로 클릭한 횟수를 분석 |
| 푸시메시지 클릭수 | PSH | 사용자가 수신되는 푸시 메시지를 클릭한 횟수를 분석 | |
| 장바구니 상품 분석 | SCRT | 사용자가 장바구니에 담은 상품을 분석 |
